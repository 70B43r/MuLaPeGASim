OCRImgExtCounter Fields

The fields of the OCRImgExtCounter class are listed below. For a complete list of OCRImgExtCounter class members, see the OCRImgExtCounter Members topic.

Protected Instance Fields

dx Describes how often a column should be extracted. Default == 1 => every column. A value of 2 means that every 2nd column would be used for extraction
dy Describes how often a row should be extracted. Default == 1 => every row. A value of 2 means that every 2nd row would be used for extraction
threshold (inherited from OCRImageExtractor) The threshold for extracting bitmap values b/w. Default 0.5 if(pixel.brightness > threshold) pat = 1; else pat = 0
