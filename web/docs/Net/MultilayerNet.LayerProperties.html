Layer Properties

The properties of the Layer class are listed below. For a complete list of Layer class members, see the Layer Members topic.

Public Instance Properties

inputProp The input of the layer, which is calculated just in time.
Length Returns the number of neurons in the layer
neuronsProp The array of neurons in the layer
outputProp The output of the layer, which is calculated just in time.
prevLayerProp The previous layer of the current layer (this)
thresholdLayerProp The threshold vector of the neurons in the layer; thresholdLayerProp[index neuron in layer, from left to right]
weightLayerProp The weight matrix of the neurons in the layer (gets incl. the bias) weightLayerProp[index neuron in layer, from left to right][weight presynapt. neuron]
