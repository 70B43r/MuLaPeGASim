Neuron Properties

The properties of the Neuron class are listed below. For a complete list of Neuron class members, see the Neuron Members topic.

Public Instance Properties

actFuncProp The activation function of the neuron
inputProp Returns the input without the bias on-state, sets the input without overwriting the bias on-state
inputWithBiasProp Returns the input with the bias and sets the input including the bias on-state (last value in the vector)
netProp Returns the net output of the neuron.
outputProp Returns the output of the neuron, which depends on the current activation function
thresholdProp The threshold of the neuron. The negative of the last value in the vector -> -weightWithBiasProp[weightWithBiasProp.Length-1]
weightProp Returns the weight without the bias and sets the weight without overwriting the bias
weightWithBiasProp Returns the weight with the bias and sets the weight including the bias (last value in the vector)
